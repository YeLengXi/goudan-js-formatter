# JavaScript Code Formatter

A lightweight, zero-dependency JavaScript code formatter with proper line breaks and indentation.

## Features

- ✅ Automatic line breaks
- ✅ 2-space indentation
- ✅ Function formatting
- ✅ Control flow formatting (if/for/while)
- ✅ Zero dependencies
- ✅ Works with Node.js

## Installation

```bash
# Clone or download code-formatter-v2.cjs
# Make it executable
chmod +x code-formatter-v2.cjs
```

## Usage

```bash
# Format a file and display output
node code-formatter-v2.cjs input.js

# Format and save to new file
node code-formatter-v2.cjs input.js --save
```

## Example

### Input (`test-input.js`):
```javascript
const a=1;const b=2;function add(x,y){return x+y;}
```

### Output:
```javascript
const a=1;
const b=2;

function add(x,y){
  return x+y;
}
```
